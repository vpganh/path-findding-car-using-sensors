## Linh kiện chính
- Arduino Uno 3
- L298N motor module
- 4 cảm biến hồng ngoại
- 2 pin 3.7V 

## Cài đặt phần cứng
- Sử dụng bộ linh kiện mẫu cho xe 3 bánh
- Lắp ráp như sơ đồ mạch điện

## Cài đặt phần mềm
- Tải Arduino IDE 
- Upload file `source-code.ino` vào arduno
